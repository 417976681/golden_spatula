# \# Golden Spatula 项目介绍

# 

# Golden Spatula 是一个基于 C++ 和 Qt 技术栈开发的桌面程序项目。核心功能主要围绕英雄角色、属性管理和元素模板的创建与拓展。项目通过 Qt 工程和 CMake 构建，支持主流平台的开发与部署，拥有开放的代码结构和良好的代码规范基础。

# 

# \## 主要功能特色

# 

# \- 英雄信息与属性管理：通过 `hero`、`basic\_attribute` 等模块实现对角色及其基础属性的建模。

# \- 元素与模板系统：支持扩展元素模板，提升项目的可扩展性和复用性。

# \- 图形界面支持：采用 Qt UI 架构，拥有丰富的界面交互体验。

# \- 高度模块化设计：包括主控程序 (`main.cpp`)、核心逻辑 (`golden\_spatula.cpp/h`) 和可复用的功能模块。

# 

# \## 技术实现

# 

# \- \*\*主语言\*\*：C++

# \- \*\*界面框架\*\*：Qt（.pro/.ui 文件）

# \- \*\*自动化构建\*\*：CMakeLists.txt 管理项目构建

# \- \*\*项目结构\*\*：包含源码、资源截图、工程构建与文档笔记

# 

# \## 应用场景

# 

# 适用于桌面端的策略管理类工具或小游戏原型开发，对人物信息、属性扩展与元素模板有较强需求的应用环境。
